# BIOS 设置

- SATA Operation: AHCI
- CFG Lock: Disabled
- Fast Boot: Disabled（建议)
- Secure Boot: Disabled
- Resume By PCI-E/Networking Device: 如果要睡眠就关，要远程唤醒就开
- Resume By Intel Onboard LAN: 如果要睡眠就关，要远程唤醒就开
